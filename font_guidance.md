
# cnsnt Typography Guidelines

## 1. Font Families
- **Primary (UI Text):**
  - System Sans-Serif: SF Pro / Roboto / Segoe UI
  - Fallback: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif

## 2. Size Scale (px)
| Style           | Size | Line Height |
|-----------------|------|-------------|
| H1 / Title      | 32   | 40          |
| H2 / Section    | 24   | 32          |
| Body            | 16   | 24          |
| Label           | 14   | 20          |
| Caption         | 12   | 16          |

## 3. Weights
- Bold (700): H1, CTA
- Medium (500): H2, labels
- Regular (400): body

## 4. Colors
- Dark Mode Text: #EEEEE6 on #121212
- Light Mode Text: #222222 on #FFFFFF
- Accent Mint Teal: #3DDC97

## 5. Accessibility
- Ensure 4.5:1 contrast for body text, 3:1 for large text.
